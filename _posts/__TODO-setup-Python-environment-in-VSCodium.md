### Step 1: Activate your env environment

```bash
source env/bin/activate
```
**How to tell it worked:** Your terminal prompt will change and show `(env)` at the very beginning, looking something like this: `(env) rogelio@Dell-OptiPlex:~/Dropbox/03-ITC-2024-materias/...$`


### Step 2: Install Pandas

Now that the environment is active, any package you install will stay isolated inside `env`. Run:


```bash
pip3 install packagename
```
